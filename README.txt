Hello,
you will need to install pygame, pysat and sympy libraries:

pygame- https://www.pygame.org/download.shtml
pysat- https://pysathq.github.io/installation/
sympy- https://www.sympy.org/en/download.html

If you have issues you may need to run an environment with an older version of python
(I use Python 3.9) in your terminal.

References for code:
Part of the pygame grid drawing algorithm was inspired by reddit user mala-22
